# Security

Annotated Media Fix is designed to run locally.

## Data Handling

- Annotation JSON stays on the user's machine unless they paste or upload it elsewhere.
- The example scripts do not call external APIs.
- The review canvas does not need a backend server.

## Reporting Issues

If you find a security issue, please open a private report through GitHub's security advisory flow if available, or contact the repository owner directly.

Do not include secrets, customer media, private screenshots, or credentials in public issues.
