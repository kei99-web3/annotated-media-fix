# Example User Feedback

This example shows how a reviewer can write natural Japanese feedback while marking the image.

## Feedback Notes

### a01

文字がずれているので、円の真ん中に文字をちゃんと置いてください。

### a02

四角に対して文字がずれています。
四角を文字に合わせて小さくしてください。

### a03

このエリアが少し寂しいので、何か可愛らしいイラストを足しておいて。

### a04

この辺の書体とか、もう少し何かかっこよくしておいてほしい。
全体のトーンを合わせつつ、背景の色も変えておいて。

## How The Agent Interprets It

| ID | Interpreted Fix |
| --- | --- |
| a01 | Center the text inside the circular badge. |
| a02 | Shrink the blue label to fit the text better. |
| a03 | Add a small cute accent in the empty upper-right area. |
| a04 | Improve typography and adjust the background tone while preserving the layout. |
